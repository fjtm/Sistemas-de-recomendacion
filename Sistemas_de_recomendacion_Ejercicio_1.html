Tf-idf¶

Para crear el sistema de recomendacin basado en el contenido utilizarmos el mtodo Tf-idf que determina la importancia de un trmino (palabra o conjunto de palabras) en un documento dentro del contexto de un confunto de documentos. Matemticamente se puede expresar como:

$$ tf-idf_{i,j} = tf_{i,j} \bullet log(\frac{N}{df_i})$$

Con:

  • $tf_{i,j}$ : el nmero total de aparaciones de i en j (frecuencia del trmino).

  • $df_i$ : el nmero total de documentos en los que aparece i.

  • $N$ : el nmero total de documentos.

    Para ello podemos utilizar la funcin TfidfVectorizer de sklearn. Esta tiene los siguientes parmetros principales:

  • analyzer : Determina el formato de las variables a utilizar. En concreto en nuestro caso utlizaremos palabras.

  • stop_words : Elimina palabras que no son tiles para la recomendacin, como son por ejemplo and, the, him...

  • min_df : Determina el mnimo nmero de veces que un trmino debe aparecer para considerarse.

  • max_df : Determina el mximo nmero de veces que un trmino puede aparecer antes de descartarse.

  • ngram_range : Se utiliza para indicar el nmero de combinaciones de palabras a tener en cuentas. Es decir, si utilizamos ngram_range = (1,3), considerar las palabras individuales, pares de palabras y trios de estas. As, se tendr en cuenta cualquiera de estas combinaciones si se repite un mnimo de veces (min_df).

    • Implementamos el mtodo con los parmetros que se muestran en la funcin TfidfVectorizer.

    In [15]:
    def recommendations(title, choose = "cosine similarity"):
    idx = indices[title]
    if choose == "cosine similarity":
        sim_scores = list(enumerate(cosine_sim[idx]))
    elif choose == "gaussian kernel":
        sim_scores = list(enumerate(kernel_pairw[idx]))
    elif choose == "euclidean distance":
        sim_scores = list(enumerate(eucl_dist[idx]))
    elif choose == "manhattan distance":
        sim_scores = list(enumerate(manh_dist[idx]))
    sim_scores = sorted(sim_scores, key=lambda x: x[1], reverse=True)
    sim_scores = sim_scores[1:21]
    movie_indices = [i[0] for i in sim_scores]
    return titles.iloc[movie_indices]
